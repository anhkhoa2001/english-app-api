             Hệ thống học tiếng anh trực tuyến trên nền tảng web
Đối tượng: Học viên, Giảng viên, Quản trị viên
Danh sách chức năng: 
1. Học viên
- Đăng nhập, đăng ký dưới tư cách học viên
- Quán lý khóa học online
    + Đăng ký khóa học
    + Xem bài giảng, làm minitest thực hành
    + thảo luận về khóa học
- Quản lý thư viện đề thi
    + Đăng ký đề thi
    + Thi online
    + Xếp hạng theo sao + thảo luận về đề thi
- Quản lý thư viện tài liệu
    + Thêm sửa xóa tài liệu
    + Xem online hoặc tải xuống tài liệu
    + Xếp hạng sao và đánh giá tài liệu
- Quản lý blog
    + Thêm sửa xóa blog
    + Thảo luận về blog
2. Giảng viên
- Đăng nhập, đăng ký dưới tư cách giảng viên
- Quản lý khóa học online
    + Thêm, sửa, xóa khóa học như tạo khóa học, thêm minitest, video ...
    + Tham gia thảo luận về khóa học cùng học viên
- Quản lý thư viện đề thi
    + Phân loại đề thi theo chủ đề, trình độ ...
    + Thêm sửa xóa đề thi + đáp án
    + Xem lịch sử thi của từng học viên
- Quản lý thư viện tài liệu
    + Chia sẻ tài liệu dưới dạng pdf
    + tải xuống tài liệu
    + Thảo luận với học viên về tài liệu
- Quản lý blog
    + Thêm sửa xóa blog
    + Thảo luận về blog
3. Quản trị viên
- Quản lý học viên, giảng viên
- Quản lý khóa học
- Quản lý đề thi
- Quản lý tài liệu
- Xem báo cáo, thống kê như người đăng ký, số lượt tải tài liệu ....
- Duyệt blog của học viên, giảng viên
4. Một số tính năng nâng cao
- Đánh giá khóa học, chat giữa các học viên (Sử dụng socket hoặc kafka để cho phép chat realtime)
- Lưu trữ file lên aws
- Đăng nhập bằng google, github


**Công nghệ:
- FE: ReactJS
- BE: Spring Boot + Database
